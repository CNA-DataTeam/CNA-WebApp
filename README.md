# CNA WebApp

Internal Streamlit application for Clark National Accounts logistics workflows.

## What It Includes
- Task tracking and analytics
- Task and user administration
- Packaging estimation
- Time allocation entry and export
- FedEx address validation review
- Stocking agreement generation

## Run It
1. Make sure Python 3.11 is installed on Windows.
2. Run `setup.bat` once to create `.venv` and install dependencies.
3. Run `StartApp.bat` to launch the app in your browser.

## Notes
- The application depends on internal shared files, network locations, and synced business data.
